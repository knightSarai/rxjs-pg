When to use from versus of

  • Use from when whatever you're working with is kind of like an observable
    • Array
    • Promises
    • Object with a scubscribe method (a.k.a have a similar API)
  • Use of when it's nothing like an observable
    • String
    • Number
    • Object

Two Ways of Listenings to Observables

If we only care about the next values, we can use subscribe method 
          
            example$.subscrib(value => console.log(value));
If we want explicitly care about either the error or when the obeservable is complete, we can use subscribe method with three callbacks 
          
            example$.subscrib({
              next: value => console.log(value),
              error: err => console.log(err),
              complete: () => console.log('complete')
            });
How obeservables are completed, There's are a genrator under the hood, and when the generator is done, the observable is complete
